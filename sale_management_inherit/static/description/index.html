Quotation template pricelists

This application allows you to manage pricing based on the quotation templates, using the selected pricelist in the template

In order to use this feature, you need to do the following steps:

  1. Install the module Quotation template pricelist.
  2. Make sure to enable quotation template feature in the sale's settings.
  3. Create and assign pricelist for each quotation template you have.

The Price list will be applied automatically based on the quotation template selected in the sale order.
